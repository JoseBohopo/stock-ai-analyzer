# Stock AI Analyzer

**Analyze your stocks with AI**.  
Add up to **3 stock tickers**, apply filters, access real-time data from [Polygon.io](https://polygon.io/), and get insights, trends, and analysis generated by ChatGPT (using your own API key).

## ğŸ› ï¸ Installation & Setup

1. **Clone this repository**
   ```sh
   git clone https://github.com/JoseBohopo/stock-ai-analyzer.git
   cd stock-ai-analyzer
   ```

2. **Install dependencies**
   ```sh
   npm install
   ```

3. **Set up your environment variables**

   Rename the `.env.example` file to `.env` and add your API keys:

   ```env
   VITE_POLYGON_API_KEY=YOUR_POLYGON_API_KEY
   VITE_OPENAI_API_KEY=YOUR_OPENAI_API_KEY
   ```

   - Get your Polygon.io API key at [https://polygon.io/dashboard/api-keys](https://polygon.io/dashboard/api-keys)
   - Get your OpenAI API key at [https://platform.openai.com/api-keys](https://platform.openai.com/api-keys)

4. **Run the app**
   ```sh
   npm run dev
   ```
   Then open [http://localhost:5173](http://localhost:5173) in your browser.
